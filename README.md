# 🍭 Nassau Candy Profitability Dashboard

A professional interactive business analytics dashboard built using **Streamlit**, **Plotly**, and **Python** to analyze product profitability, gross margins, revenue, and sales performance for Nassau Candy Distributor.

---

## 📌 Project Overview

This dashboard provides an interactive view of Nassau Candy's business performance by allowing users to explore profitability across products and divisions.

The project helps identify:

- Top profitable products
- Gross margin performance
- Revenue contribution
- Monthly profitability trends
- Division-wise performance
- Pareto analysis

---

## ✨ Features

- Interactive KPI Cards
- Product Profitability Analysis
- Division Performance Dashboard
- Cost & Margin Diagnostics
- Pareto Analysis
- Product Search Filter

## 🚀 How to Run

Clone the repository

```
git clone <repository-link>
```

Install dependencies

```
pip install -r requirements.txt
```

Run the application

```
streamlit run app/streamlit_app.py
```
